================================================================================
                      DATABASE SPECIFICATION
================================================================================

DATABASE FILE: ~/.task-manager/tasks.db (SQLite3)

SCHEMA:
  TABLE: tasks
    - id INTEGER PRIMARY KEY AUTOINCREMENT
    - title TEXT NOT NULL
    - description TEXT DEFAULT ''
    - project TEXT DEFAULT 'General'
    - priority TEXT DEFAULT 'medium' (CHECK: high, medium, low)
    - status TEXT DEFAULT 'pending' (CHECK: pending, in_progress, done)
    - due_date TEXT
    - created_at TEXT NOT NULL (ISO format timestamp)
    - completed_at TEXT

INDEXES:
  - idx_project ON tasks(project)
  - idx_status ON tasks(status)

CONSTRAINTS:
  - priority CONSTRAINT (high|medium|low)
  - status CONSTRAINT (pending|in_progress|done)

MANAGEMENT:
  Auto-created on first run
  Location: ~/.task-manager/tasks.db
  Size: Minimal (kilobytes)
  Reset: rm ~/.task-manager/tasks.db
